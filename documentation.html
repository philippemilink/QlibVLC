QlibVLC()

Constructeur de la classe QlibVLC.

Utilisation: 

QlibVLC* vlc = new QlibVLC();

Haut

void setMedia(QString path)

Sélectionne un fichier audio.

Paramètre:

  • path: le chemin du fichier audio. Le type de chemin (avec des slashs, des anti-slashs ou des doubles slashs) ne devrait normalement pas poser de problème...

Utilisation: 
vlc->setMedia("C:\Users\Admin\Music\Hit.mp3");
Lorsque l'utilisation du fichier est terminée, ou avant d'utiliser un autre fichier, il est bon de l'éjecter proprement en utilisant ejectMedia().

Haut

void ejectMedia()

Ejecte le media actuellement utilisé par QlibVLC.
Si le fichier est actuellement en lecture, il est stoppé.

Utilisation: 

vlc->eject_media();

Haut

void play()

Met en lecture le media actuellement utilisé par QlibVLC.

Utilisation: 

vlc->play();

Haut

void pause(bool pause)

Met en pause ou reprend la lecture du media actuellement utilisé par QlibVLC.

Paramètre:

  • pause: si ce booléen vaut true, alors la fonction enclanche la pause et si ce booléen vaut false, la fonction enlève la pause.

Utilisation: 
vlc->pause(true); // Met en pause.

Haut

bool isPlaying()

Indique si le media est en lecture ou non.

Valeur de retour:
Un booléen. true: lecture en cours, sinon false.

Utilisation: 

if(m_vlc->isPlaying())
	label->setText("Lecture en cours..."); // Un label précedemment créé.
else
	label->setText("Plus de lecture.");

int trackNumber()

Retourne le numéro de la piste du média.

Valeur de retour:
Le numéro de piste. -1 si le numéro est inconnu.

Utilisation: 

int trackNumber = m_vlc->trackNumber();
label->setText("Numéro de piste: " + QString::number(trackNumber));

QString title()

Retourne le titre du média.

Valeur de retour:
Le titre du média, sous format QString. Si le titre n'est pas précisé dans les attributs du média (tags ID3...), la valeur retournée est le nom du fichier sans l'extension.

Utilisation: 

QString titre = vlc->title();
label->setText("Titre: " + title);

Haut

int duration()

Retourne la durée du média actuellement utilisé par QlibVLC.

Valeur de retour:
La durée du média, en millisecondes.

Utilisation: 

int duree = vlc->duration();
label->setText("Durée: " + QString::number(duree / 1000) + "s");

Haut

int getDuration(QString path)

Retourne la durée du média, dont le chemin est passé en paramètre.

Paramètre:

  • path: le chemin du fichier audio. Le type de chemin (avec des slashs, des anti-slashs ou des doubles slashs) ne devrait normalement pas poser de problème...

Valeur de retour:
La durée du média, en millisecondes.

Utilisation: 
int duree = vlc->getDuration("C:\Users\Admin\Music\Hit.mp3");
label->setText("Durée: " + QString::number(duree / 1000) + "s");

Haut

std::vector<QlibVLCOutput> getOutputs()

Retourne les sorties audio disponibles sur l'ordinateur.

Valeur de retour:
Un tableau vector de structures QlibVLCOutput.
Cette structure possède deux variables: name, de type QString, qui est le nom de la sortie, mais plus sous la forme d'un identifiant; et description, de type QString, qui est une description de cette sortie audio, mais en réalité, il s'agit d'un nom plus compréhensible pour l'utilisateur.

Utilisation: 

std::vector<QlibVLCOutput> outputs = vlc->getOutputs();
comboBoxOutputs->addItem("No specified");
for(int i=0; i<outputs.size(); i++)
{
	comboBoxOutputs->addItem(outputs[i].description);
}

int getVolume()

Le volume, compris entre 0 (muet) et 100.

Utilisation:

Signaux - Détails

void positionChanged(int newPosition)

Signal émis lorsque la position dans le fichier audio change.

Variable transmise:

  • value: la nouvelle position dans le média, en millisecondes.

Utilisation: 
// Dans le constructeur d'un player (par exemple !):
connect(vlc, SIGNAL(positionChanged(int)), this, SLOT(timeChanged(int)));

// Le slot timeChanged:
void Player::timeChanged(int newTime)
{
    valuePlayedTime->setText("Position: " + QString::number(newTime/1000) + "s");
}

Haut

void endTrack()

Signal émis lorsque la lecture du fichier audio est terminée.

Utilisation: 

// Dans le constructeur d'un player (par exemple !):
connect(vlc, SIGNAL(endTrack()), this, SLOT(endPlaying()));

// Le slot endPlaying:
void Player::endPlaying()
{
    vlc->ejectMedia();

}
