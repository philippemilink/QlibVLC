Installation de QlibVLC sous Windows

QlibVLC est en réalité une classe C++ qui utilise libVLC. QlibVLC a donc comme dépendance libVLC.

Installation de libVLC

Commencez par installer VLC Media Player, si ce n'est pas déjà fait. Pendant l'installation, cochez que vous souhaitez également le kit de développement (peut-être nommé SDK).

Dans le dossier de VLC (ressemblant à C:\Programmes\VideoLAN\VLC\), allez dans le dossier sdk.
Copiez tout le contenu du dossier include dans le dossier include de votre compilateur.
De la même façon, copiez tout le contenu du dossier lib dans le dossier lib de votre compilateur.

Ajout de QlibVLC à votre projet Qt

Dans votre projet Qt, ajoutez les fichiers QlibVLC.cpp et QlibVLC.h.

Ajoutez la bibliothèque de libVLC au linker de votre projet.
Pour cela, faîtes un clic droit sur votre projet dans l'arborescence de Qt Creator, cliquez sur Ajouter une bibliothèquee.
Sélectionnez Bibliothèque externe.
Dans le Fichier de bibliothèque, sélectionnez le fichier libvlc.lib, dans le dossier lib de votre compilateur.
Décochez les autres plateformes que Windows et Ajouter le suffixe "d" pour la version debug.
Suivant... Terminer...

Vous n'avez plus qu'à inclure QlibVLC dans votre code: #include "QlibVLC.h"

Quelques DLL...

Pour que la compilation fonctionne, il faut ajouter deux DLLs dans le dossier racine de votre projet.
Ces DLLs sont libvlc.dll et libvlccore.dll. Elles sont présentes dans le dossier du programme VLC.
Il faut également copier le dossier plugins du dossier de VLC dans le dossier racine de votre projet. Le dossier est assez lourd, toutes les DLLs présentes dedans ne sont pas nécessaires pour faire fonctionner votre programme. Vous pouvez essayer de trouver les inutiles (celles qui, quand on les supprime, empêchent l'éxecution du programme) et les supprimer.
