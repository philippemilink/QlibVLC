Installation de QlibVLC sous Linux

QlibVLC est en réalité une classe C++ qui utilise libVLC. QlibVLC a donc comme dépendance libVLC.

Installation de libVLC

libVLC n'est pas installé par défaut avec VLC sur Linux.
Il faut installer le paquet libvlc-dev: sudo apt-get install libvlc-dev

Ajout de QlibVLC à votre projet Qt.

Dans votre projet Qt, ajoutez les fichiers QlibVLC.cpp et QlibVLC.h.

Ajoutez la bibliothèque de libVLC au linker de votre projet.
Pour cela, faîtes un clic droit sur votre projet dans l'arborescence de Qt Creator, cliquez sur Ajouter une bibliothèque (ou Add library).
Sélectionnez Bibliothèque externe (ou External library).
Indiquer dans le chemin de la bibliothèque le fichier /usr/lib/libvlc.a.
Décochez toutes les autres plateformes que Linux.
