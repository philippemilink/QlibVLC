Téléchargements

Archive ZIP

Cette archive ZIP est en réalité tout le contenu du projet présent sur GitHub. Ceci inclut la bibliothèque en elle-même et un projet qui sert d'exemple.
Si vous souhaitez utiliser QlibVLC dans votre projet, vous n'avez besoin que des fichiers QlibVLC.cpp et QlibVLC.h. Suivez ensuite le tutoriel d'installation.

Depuis GitHub

Vous pouvez vous rendre sur la page GitHub du projet et utiliser les fichiers qui vous intéressent.
Sinon, vous pouvez directement cloner le projet sur votre ordinateur, à condition que Git soit installé: git clone https://github.com/philippemilink/QlibVLC.git master Si ensuite vous souhaitez ensuite mettre à jour QlibVLC, il n'y a qu'à faire dans le dossier obtenu avec la commande précédente: git pull
